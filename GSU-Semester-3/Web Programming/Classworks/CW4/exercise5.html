h2

This selector makes the contain of h2 tag has blue background color and opacity of 0.5. It means that the text inside h2 is faded

div

This selector keeps the content of div absolute with top 100px and left 200px

pos

This selector keep the position of content of pos absolute with a top of 150px

table, th, td

This element will display a border of 1px in solid red

td.bg

This element td with class name bg will have a background color of green

td.bd

This element td with class name bd will have a dashed border style

th.padding

This element th will have the padding attributes as padding-top: 20px and padding-bottom: 20px

.user1

All the elements with the classname user1 will have text alignment to right and text color as violet

#user2

All the elements with the id user2 will have text alignment as center and font style as italic
